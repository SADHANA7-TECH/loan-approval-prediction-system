# 🏦 Loan Approval Prediction System

An AI-powered Loan Approval Prediction System built using Machine Learning and Streamlit. This application predicts whether a loan application is likely to be approved or rejected based on an applicant's financial information.

## 🌐 Live Demo

https://loan-approval-prediction-system-tzjt.onrender.com

---

## 📌 Project Overview

This project uses a Decision Tree Classifier trained on a loan approval dataset to analyze customer financial details and predict loan approval status.

The goal of this project is to understand the complete Machine Learning workflow—from data preprocessing and model training to deployment as a web application.

## 📊 Dataset

Dataset: Loan Approval Prediction Dataset

Features include:

- Number of Dependents
- Education
- Self Employed
- Annual Income
- Loan Amount
- Loan Term
- CIBIL Score
- Residential Asset Value
- Commercial Asset Value
- Luxury Asset Value
- Bank Asset Value

Target Variable:

- Loan Status
  - Approved
  - Rejected

## 🤖 Machine Learning Workflow

1. Load Dataset
2. Clean Data
3. Encode Categorical Variables
4. Split Dataset into Training and Testing Sets
5. Train Decision Tree Model
6. Evaluate Model
7. Save Trained Model
8. Deploy using Streamlit

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/SADHANA7-TECH/loan-approval-prediction-system.git
```

Move into the project folder

```bash
cd loan-approval-prediction-system
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
streamlit run app.py
```
